# flet-pdfrx
FletPdfrx control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

* **Git dependency**

Link to git repository:

```
dependencies = [
  "flet-pdfrx @ git+https://github.com/MyGithubAccount/flet-pdfrx",
  "flet>=0.80.5",
]
```

* **uv/pip dependency**  

If the package is published on pypi.org:

```
dependencies = [
  "flet-pdfrx",
  "flet>=0.80.5",
]
```

Build your app:
```
flet build macos -v
```
